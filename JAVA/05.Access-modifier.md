## 접근제어자

객체 지향에서 클래스와 클래스 멤버 선언 시 키워드를 통해 사용자가 굳이 알 필요가 없는 정보를 은닉할 수 있다.

### 4 가지 접근 제어자

- 접근 제어자의 접근 범위
  - public > protected > default > private

|           | 클래스 내부 | 같은 패키지 | 자식 클래스 | 다른 패키지 |
| --------- | :---------: | :---------: | :---------: | :---------: |
| public    |      O      |      O      |      O      |      O      |
| protected |      O      |      O      |      O      |      X      |
| default   |      O      |      O      |      X      |      X      |
| private   |      O      |      X      |      X      |      X      |
