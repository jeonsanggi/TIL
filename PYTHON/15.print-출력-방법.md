## print 출력 방법

```python
print(1, 2, 3)
print("%d %d %d" %(1, 2, 3))
print("%f %.1f %.2f" %(1.1, 2.2, 3.2))
print("%s--%s--%s" %('abc', 'def', 'ghi'))
print("{} {} {}".format('abc', 'def', 'ghi'))
print("{1}, {0}, {2}".format('abc', 'def', 'ghi'))
print(f"{'def'}{'abc'}{'ghi'}")
print("{0:>10s}".format('python'))
print("{0:<10s}".format('python'))
print(f"{'python':>10s}")
print(f"{'python':<10s}")
```
